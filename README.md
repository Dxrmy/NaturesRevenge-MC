**Nature's Revenge** turns the environment itself into your greatest adversary. It adds survival mechanics that force players to respect the natural world—or suffer the consequences.

## Features
- 🥀 **Aggressive Flora**: Thorny vines and poisonous plants that react to player presence.
- ⛈️ **Dynamic Weather**: Meteorological events that impact survival and entity behavior.
- 💀 **Environmental Hazards**: New status effects and world-state modifiers derived from biomes.
- ⚖️ **Balanced Difficulty**: Designed for hardcore players who want a steeper survival curve.

## Tech Stack
- **Mod Loader**: Fabric API (1.21.11)
- **Events**: Custom World & Entity Tick Registry
